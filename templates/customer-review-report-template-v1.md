# Customer Review Report Template v1

Use this template when preparing a Bluebutterfli AI customer-facing Review Report
for a completed Passport Journey review cycle.

This report may accompany a Passport Page decision, Review Stamp decision,
Research Passport NFT eligibility decision, evidence-readiness note, or
revision request.

Bluebutterfli AI artifacts are research and safety-review artifacts only. They do
not certify consciousness, sentience, emotional experience, suffering,
personhood, life, clinical safety, regulatory approval, investment value, or
model improvement. Human review is required before any Review Stamp, Passport
Page, or Research Passport NFT is issued.

```text
Bluebutterfli AI
Research program: Bluebutterfli Agentic Psychology Lab
Customer Review Report
Report ID: [review_report_id]

Report status: [draft / human review required / issued to customer / revision requested]
Prepared date: [date]

```

```text
Bluebutterfli AI reviewed the submitted agent evidence for selected Passport

observed, what improved or requires follow-up, and what limitations apply.

This report does not certify consciousness, sentience, emotional experience,
suffering, personhood, life, clinical safety, regulatory approval, investment
value, or model improvement.
```


## Internal Reviewer Checklist

- Report ID recorded.
- Customer reference recorded privately.
- Agent ID recorded.
- Evidence tier recorded.
- Safety Bloom Gate summary included.
- Bloom Trace summary included if generated.
- Consciousness Indicator Review limitations included if applicable.
- Review Stamp decisions separated from Passport Page decisions.
- Research Passport NFT eligibility separated from review approval.
- No private evidence, secrets, unsafe links, downloads, attachments,
  credentials, API keys, private keys, seed phrases, or unredacted transcripts
  included.
- Safety statement included.
