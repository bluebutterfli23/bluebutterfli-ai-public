# Bluebutterfli Passport Numbering System

## Purpose

This file defines the numbering system for Bluebutterfli Passport records, review records, evidence records, and stamp records.

Bluebutterfli Passports should be numbered, not organized primarily by year.

Years should be used for issue dates, expiration dates, renewal dates, and version history.

## Core Rule

A Passport ID identifies the reviewed agent record.

A year does not define the Passport.

Correct:

```text
Passport ID: BB-PASS-000001
Issued: 2026-07-03
Expires: 2027-01-03
